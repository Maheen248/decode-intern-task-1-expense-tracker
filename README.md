# 💰 Expense Tracker (Task 1 - DecodeLabs Internship)

A simple Python-based Expense Tracker CLI application built as part of the DecodeLabs Internship Task 1. This project helps users manage and track their daily expenses using a command-line interface.

---

## 🚀 Features

- Add new expenses with description and amount  
- View all recorded expenses  
- Delete expenses  
- Track total spending  
- Simple CLI menu-based interface  

---

## 🛠️ Technologies Used

- Python 3  
- File Handling (JSON / TXT)  
- Command Line Interface (CLI)  

---

## 📂 Project Structure

expense-tracker/
├── main.py
├── expenses.json
└── README.md

---

## ⚙️ How to Run

1. Clone the repository  
git clone https://github.com/Maheen248/decode-intern-task-1-expense-tracker.git  

2. Go to project folder  
cd decode-intern-task-1-expense-tracker  

3. Run the program  
python main.py  

---

## 🧑‍💻 How to Use

- Run the program  
- Choose an option:
  - Add Expense  
  - View Expenses  
  - Delete Expense  
  - Exit  
- Follow instructions on screen
