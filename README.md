# 💰 Expense Tracker (Task 1 - DecodeLabs Internship)

A simple Python-based Expense Tracker CLI application built as part of the DecodeLabs Internship Task 1. This project helps users manage their daily expenses using a command-line interface.

## 🚀 Features

* Add new expenses with description and amount
* View all recorded expenses
* Delete expenses
* Track total spending
* Simple CLI-based interface

## 🛠️ Technologies Used

* Python 3
* File Handling (JSON/TXT)
* Command Line Interface (CLI)
