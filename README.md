
# 💰 Expense Tracker (Task 1 - DecodeLabs Internship)

A simple Python-based Expense Tracker CLI application built as part of the DecodeLabs Internship Task 1.  
This project helps users manage and track their daily expenses efficiently using a command-line interface.

## 🚀 Features

- Add new expenses with description and amount  
- View all recorded expenses in a structured format  
- Delete unwanted expense entries  
- Track total spending  
- Simple and interactive CLI menu  

## 🛠️ Technologies Used

- Python 3  
- File Handling (JSON / TXT depending on implementation)  
- Command Line Interface (CLI)
